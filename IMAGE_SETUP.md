# إعداد الصور للتطبيق

## الصور المطلوبة

يجب إضافة الصور التالية إلى **Assets.xcassets** في مشروع iOS:

### 1. `espresso_cup`
- **الوصف**: صورة كاسة إسبريسو (كاسة ورقية بيضاء)
- **الحجم المقترح**: 120x120 نقطة @2x (240x240 بكسل)
- **التنسيق**: PNG أو JPG

### 2. `turkish_coffee_cup`
- **الوصف**: صورة كاسة قهوة تركية (كاسة حمراء)
- **الحجم المقترح**: 120x120 نقطة @2x (240x240 بكسل)
- **التنسيق**: PNG أو JPG

### 3. `turkish_coffee_packet`
- **الوصف**: صورة باكيت قهوة تركية (الباكيت الأبيض)
- **الحجم المقترح**: 120x120 نقطة @2x (240x240 بكسل)
- **التنسيق**: PNG أو JPG

### 4. `coffee_cup` (اختياري)
- **الوصف**: صورة عامة للكاسات الأخرى (Latte, Cappuccino, etc.)
- **الحجم المقترح**: 120x120 نقطة @2x (240x240 بكسل)
- **التنسيق**: PNG أو JPG

### 5. `coffee_packet` (اختياري)
- **الوصف**: صورة عامة للباكيتات الأخرى (إسبريسو بالوزن، أمريكان، etc.)
- **الحجم المقترح**: 120x120 نقطة @2x (240x240 بكسل)
- **التنسيق**: PNG أو JPG

## خطوات الإضافة

1. افتح Xcode
2. اذهب إلى **Assets.xcassets**
3. اضغط زر **+** لإنشاء Image Set جديد
4. اسم الصورة: `espresso_cup` (بدون مسافات، كل كلمة بخط صغير)
5. اسحب الصورة إلى الـ Image Set
6. كرر الخطوات للصور الأخرى

## المنطق الحالي

الكود يحدد الصورة تلقائياً بناءً على نوع المنتج:

- **قهوة تركية بالوزن** (1kg, 500g, 250g) → `turkish_coffee_packet`
- **كاسة قهوة تركية** (Brewed category) → `turkish_coffee_cup`
- **كاسة إسبريسو** → `espresso_cup`
- **كاسات أخرى** (Latte, Cappuccino) → `coffee_cup`
- **منتجات أخرى بالوزن** → `coffee_packet`

## ملاحظات

- إذا لم يتم العثور على الصورة، سيظهر placeholder رمادي مع أيقونة `photo`
- الصور ستظهر بجانب كل منتج في صفحة تفاصيل الطلب
- حجم الصورة: 60x60 نقطة (120x120 بكسل @2x)
